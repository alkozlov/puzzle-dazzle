# Новые виды лабиринтов

1. Polar/Circular Maze
- Description: A maze with a circular layout, where paths radiate outward from a central point. The maze is divided into concentric rings, and the user must navigate from the center to the outer edge (or vice versa). This type of maze offers a unique visual style and a different navigation challenge compared to traditional rectangular mazes.
