# Улучшение граифической визуализации лабиринта

Гавная цель данного изменения - улучшить графическую визуализацию лабиринта, чтобы сделать его более понятным и привлекательным для пользователей. Текущий дизайн лабиринта представлен на скриншоте `./current_maze_design.jpg`. Этот дизайн является максимально простым и может не вызывать интерес у целевой аудитории (дети и подростки).

Предлагаемый дизайн представлен на скриншоте `./proposed_maze_design.png`. В новом дизайне используется цвет для линий лабиринта, немного увеличин вес линий, а соединения сделаны более плавными. Это делает лабиринт более привлекательным и легким для понимания, особенно для детей.

## Основные изменения

1. **Цвет линий**: В новом дизайне линии лабиринта окрашены в определенный цвет. На данном этапе этот параметр должен быть вынесен где-то  в код. Пользователь не сможет изменять цвет. Лучше всего постараться использовать цвет линий из изображения с примером дизайна, который представлен в `./proposed_maze_design.png`.

2. **Толщина линий**: Линии лабиринта стали немного толще, что делает их более заметными и улучшает визуальное восприятие. Этот параметр также должен быть вынесен в код, чтобы его можно было легко изменить в будущем при необходимости.

3. **Плавные соединения**: В новом дизайне соединения между линиями лабиринта сделаны более плавными, что придает лабиринту более современный и привлекательный вид.

4. **Использование большего пространства**: Новый дизайн лабиринта должен занимать больше пространства на экране. Это потребует доработки или переработки текущего UI-элемента, который отображает лабиринт. Цель - сделать лабиринт более заметным и привлекательным для пользователей.

## Ожидаемые результаты
- Улучшение визуальной привлекательности лабиринта, что может увеличить вовлеченность пользователей.
- Повышение удобства использования, особенно для детей, которые являются основной целевой аудиторией приложения.
