# Упрощение интерфейса

Цель данного change request - упростить интерфейс для пользователей, уменьшив количество страниц до 2.

## Текущая структура

- **Main screen**
- **Settings screen**
- **Maze generation screen**

## Предложенные изменения

- **Maze generation screen** будет главной страницей. При первом открытитии приложения пользователь сразу попадает на эту страницу. Однако лабиринт не будет генерироваться автоматически. Пользователю нужно будет нажать соответствующую кнопку, чтобы начать генерацию лабиринта. Это позволит пользователю сразу увидеть результат и не отвлекаться на лишние настройки.

- **Settings screen** остается без изменений.

Таким образом, мы сохраняем функциональность приложения, но упрощаем навигацию и уменьшаем количество шагов для пользователя. Это может сделать приложение более интуитивным и удобным для широкого круга пользователей.

## Toobar

Новая главная страница будет включать в себя тулбар (или что-то похожее) в нижней части экрана, который будет содержать следующие кнопки:

1. **Start** — для начала генерации лабиринта.
2. **Settings** — для перехода на страницу настроек.
3. **Save** — для сохранения сгенерированного лабиринта (активируется только после генерации).
4. **Print** — для печати сгенерированного лабиринта (активируется только после генерации).

Все кнопки должны использовать иконки для более интуитивного понимания, а также иметь текстовые подписи для ясности. Это обеспечит удобство использования и позволит пользователям быстро находить нужные функции.

Кнопка "Start" будет расположена в центре тулбара, так как это основное действие. Кнопки "Settings", "Save" и "Print" будут расположены по бокам для легкого доступа. Кнопка "Start" должна создавать новыйлабиринт при каждом нажатии, а кнопки "Save" и "Print" должны быть неактивными, пока лабиринт не будет сгенерирован. Это поможет избежать путаницы и обеспечит логичный поток действий для пользователя.

Страница **Settings** содержит только 1 кнопку "Back" для возвращения на страницу генерации лабиринта.
