# Концепция CR-4

На данный момент настройка генерации лабиринта осуществляется на странице `Settings`. Полльзователю доступны 3 категории настроек: размер лабиринта, сложность и форма. Все эти 3 настройки представлены в виде выпадающего списка. В рамках CR-4 предлагается изменить интерфейс страницы `Settings` и заменить выпадающие списки на наборы кнопок-переключателей (toggle-buttons). Это упростить процесс настройки генерации лабиринта (1 клиек вместо 2) и сделает его более интуитивно понятным для пользователей.

## Раземер лабиринта

Выподающееся меню для настройки размера лабиринта будет заменено на 3 кнопки-переключателя, каждая из которых соответствует одному из доступных размеров: Small, Medium и Large. Пользователь сможет выбрать только одну кнопку, которая будет выделена, указывая на выбранный размер лабиринта. На данных кнопках не желательно использовать текст, а вместо этого использовать иконки, которые будут интуитивно понятными для пользователей. Например, для Small можно использовать иконку маленького квадрата, для Medium - средний квадрат, а для Large - большой квадрат. Необходимо подобрать подходязую иконку, т.к. целевая аудитория - дети, и они должны легко понимать, что означает каждая кнопка.

## Сложность лабиринта

Выподающееся меню для настройки сложности лабиринта будет заменено на 3 кнопки-переключателя, каждая из которых соответствует одному из доступных уровней сложности: Easy, Medium и Hard. Пользователь сможет выбрать только одну кнопку, которая будет выделена, указывая на выбранный уровень сложности. На данных кнопках не желательно использовать текст, а вместо этого использовать иконки, которые будут интуитивно понятными для пользователей. Например, для Easy можно использовать иконку улыбающегося лица, для Medium - нейтрального лица, а для Hard - грустного лица. Необходимо подобрать подходязую иконку, т.к. целевая аудитория - дети, и они должны легко понимать, что означает каждая кнопка.

## Форма лабиринта

Выподающееся меню для настройки формы лабиринта будет заменено на 4 кнопки-переключателя, каждая из которых соответствует одному из доступных форм лабиринта: Rectangle, Circle, Diamond and Heart. В дальнейшем форм будет больше. Это нужно учесть при изменении разметки интерфейса. Пользователь сможет выбрать только одну кнопку, которая будет выделена, указывая на выбранную форму лабиринта. На данных кнопках не желательно использовать текст, а вместо этого использовать иконки, которые будут интуитивно понятными для пользователей. Например, для Rectangle можно использовать иконку квадрата, для Circle - иконку круга и т.д. Необходимо подобрать подходязую иконку, т.к. целевая аудитория - дети, и они должны легко понимать, что означает каждая кнопка.

## Сохранение настроек

Приложение уже запоминает выбранные настройки и применяет их при следующем запуске. Поэтому после замены выпадающих списков на кнопки-переключатели, необходимо убедиться, что выбранные пользователем настройки сохраняются и применяются при следующем запуске приложения. Это обеспечит удобство использования и позволит пользователям не настраивать параметры каждый раз при запуске приложения.
