## Концепция

Этот проект представляет из себя приложение для мобильных устройств на платформе Android и iOS. Приложение предназначено для генерации лабиринтов. Пользователь может настроить параметры генерации лабиринта, а также выбрать шаблон для его визуализации. Либо он может воспользоваться настройками по-умолчанию. Сгенерированный лабиринт можно сохранить в виде изображения, а также поделиться им с друзьями через социальные сети или мессенджеры. Приложение будет иметь простой и интуитивно понятный интерфейс, чтобы пользователи всех возрастов могли легко создавать и наслаждаться своими лабиринтами.

### Основные функции приложения:

1. Генерация лабиринтов с различными параметрами (размер, сложность, тип генерации и т.д.).
2. Выбор шаблонов для визуализации лабиринта (различные стили, цвета, текстуры и т.д.).
3. Сохранение сгенерированного лабиринта в виде изображения или pdf-файла на устройстве пользователя.
4. Возможность поделиться сгенерированным лабиринтом через социальные сети и мессенджеры.

### Целевая аудитория:

Приложение предназначено для широкого круга пользователей, включая детей, подростков и взрослых, которые интересуются головоломками, играми и творческим процессом. Оно может быть полезным для учителей, которые хотят использовать лабиринты в образовательных целях, а также для любителей головоломок, которые хотят создавать свои собственные лабиринты и делиться ими с друзьями.

### Технологический стек:

- Платформа: Android и iOS. Первая версия приложения будет разработана для Android, а затем будет адаптирована для iOS.
- Язык программирования: dotnet 10 + C#. Использование кроссплатформенного фреймворка Xamarin или MAUI для разработки приложения, что позволит использовать один код для обеих платформ. Скорее всего будет использоваться MAUI, так как он является более современным и поддерживает больше функций для кроссплатформенной разработки.
- Библиотеки и инструменты: использование библиотек для генерации лабиринтов, а также инструментов для создания пользовательского интерфейса и работы с изображениями. Предпочтение будет отдаваться open-source решениям, чтобы снизить затраты на разработку и обеспечить гибкость в выборе инструментов.
- Система контроля версий: Git для управления кодом и совместной работы над проектом.
- Система управления проектами: использование инстрмента beads для планирования задач, отслеживания прогресса и управления командой разработчиков (инструмент будет добавлен в репозиторий проекта).

### Платные функции:

Приложение будет бесплатным для загрузки и использования, но будет предлагать ограниченное количество генераций лабиринтов и шаблонов. Для доступа к неограниченному количеству генераций и шаблонов, а также для получения дополнительных функций, таких как возможность сохранять лабиринты в высоком разрешении или использовать эксклюзивные шаблоны, пользователи смогут приобрести премиум-подписку. Премиум-подписка будет предлагаться на ежемесячной или годовой основе, с возможностью бесплатного пробного периода для новых пользователей.

Пользователи, которые не хотят приобретать премиум-подписку, смогут продолжать использовать приложение с ограниченными функциями: генерировать лабиринты не более 3 раз в день и использовать базовые шаблоны для визуализации.

Регистрация пользователей и оформление подписки будут осуществляться **только через встроенные механизмы Google Play и App Store**, что обеспечит безопасность и удобство для пользователей.
